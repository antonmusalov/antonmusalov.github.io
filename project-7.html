Project Summary

I designed and developed an automated label printing system to address a critical pain point for customers struggling with integration between their ERP systems and shipping companies. The system features a Raspberry Pi 4 running custom Flask/Vue.js software, integrated with an industrial touchscreen and thermal printer in a stainless steel enclosure suitable for food production environments.

The system has been successfully running in production for 6 months under my direct support, handling thousands of shipping labels while maintaining seamless integration with Cin7 Core ERP and StarShipIT APIs. I implemented comprehensive backup systems, remote access capabilities through Tailscale VPN, and detailed documentation to ensure reliable operation.

The technical architecture includes a Flask backend managing printer communications and API integrations, while the Vue.js frontend provides an intuitive touchscreen interface. The system uses CUPS for printer management and features automated database backups, systemd services for reliability, and full remote management capabilities.

Looking ahead, I'm working on expanding this into a cloud-based solution to serve more users. This evolution will allow multiple facilities to manage their label printing needs through a centralized platform while maintaining the robust local printing capabilities of the current system. The modular design and API-first approach have positioned the system well for this expansion.
